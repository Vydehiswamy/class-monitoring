# Class Monitoring System
## Description
The Class Monitoring System is a smart classroom application designed to monitor student activities during class hours. It helps in tracking attendance, detecting mobile phone usage, analyzing student attention, and sending alerts when required.
This project is built using Python and computer vision techniques to support effective classroom management.
## Features
- Smart attendance tracking
- Mobile phone usage detection
- Student attention analysis
- Email alert system
- CSV-based data storage
- Web interface using HTML templates
## Technologies Used
- Python
- OpenCV
- Machine Learning
- Flask
- HTML / CSS
- Git & GitHub
## Project Structure
- `modules/` – core logic files
- `utils/` – helper functions
- `templates/` – HTML files
- `static/` – static assets
- `models/` – ML models
- `datasets/` – dataset files
